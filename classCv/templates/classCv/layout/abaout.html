About This Project

Welcome to the NLP Resume Classifier! This project is designed to analyze and classify resumes based on their content. Using machine learning techniques and natural language processing (NLP), the system predicts the category or field of a resume or text entered by the user.

How It Works

The system allows users to either upload a resume file (in PDF or TXT format) or manually enter text. Once submitted, the content is processed using NLP models to predict the career field, such as Information Technology, Healthcare, Engineering, and more.

Technologies Used

  • Python - for the backend logic
  • Django - for the web framework
  • Natural Language Processing (NLP) - to analyze text content
  • Machine Learning (KNN Classifier) - to predict job categories
  • PDFMiner - for extracting text from PDF files

Purpose

The primary goal of this project is to demonstrate the capabilities of NLP and machine learning in analyzing unstructured text data, such as resumes. By automating resume classification, the system can help HR professionals, job seekers, and others gain insights into career paths or improve the recruitment process.
